AccessoryAlertBlocker

supporting dependency package for alert-free low-level workflows

AccessoryAlertBlocker 用于屏蔽 accessory firmware alert，是 `iPwnder Lite (Rootless)` 这类 low-level workflow 常见的 supporting package 之一。它本身很轻量，但并不会自动让不受支持的流程变得安全。

适合人群

适合明确知道自己为什么需要这个 helper package 的用户，尤其是需要配合 `iPwnder Lite (Rootless)` 的场景。

风险提示

屏蔽提示不等于设备状态正确，也不等于流程本身安全。If the surrounding workflow is wrong, removing an alert will not fix that.

兼容说明

支持范围：当前镜像版本保留上游 `mobilesubstrate (>= 0.9.5000)` 依赖字段。

使用条件：这是 supporting package，不应被理解为完整环境配置的替代品；其余标准依赖仍应来自用户现有的软件源组合。

使用提示

  • 只在目标流程确实需要它时再安装。
  • 把它视为 helper package，而不是独立解决方案。
  • 如果你不确定为什么需要屏蔽 accessory alert，先确认上层工具链。
